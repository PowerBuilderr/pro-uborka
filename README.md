## ⚙️ Настройка формы (Formspree)

Чтобы форма отправляла заявки на email:

1. Зарегистрируйтесь на [formspree.io](https://formspree.io)
2. Создайте новую форму, укажите ваш email
3. Скопируйте ID формы (например, `xabc1234`)
4. В файле `index.html` найдите две строки:

```
   action="https://formspree.io/f/YOUR\_FORM\_ID"
   ```

5. Замените `YOUR\_FORM\_ID` на ваш реальный ID в **обоих** местах
6. Сохраните файл и загрузите на GitHub заново
