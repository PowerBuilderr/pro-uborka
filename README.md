⚙️ Настройка формы (Formspree)
Чтобы форма отправляла заявки на email:

Зарегистрируйтесь на formspree.io
Создайте новую форму, укажите ваш email
Скопируйте ID формы (например, xabc1234)
В файле index.html найдите две строки:

   action="https://formspree.io/f/YOUR_FORM_ID"

Замените YOUR_FORM_ID на ваш реальный ID в обоих местах
Сохраните файл и загрузите на GitHub заново
